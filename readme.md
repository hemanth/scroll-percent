# scroll-precent
> Get percentage of scroll.

## Install

```
$ npm install --save scroll-precent
```

## Usage

```js
const scrollPrecent = require('scroll-precent');

document.addEventListener('scroll', () => {
	console.log(scrollPrecent()); //0-100
});
```
